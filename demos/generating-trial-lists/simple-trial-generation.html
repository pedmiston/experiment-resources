Simple Trial Generation¶

counterbalance¶

One of the basic functions of trial generation is to allow for full counterbalancing. The function counterbalance takes a dict of variables and produces a pandas.DataFrame output.

labtools.trials_functions.counterbalance(conditions, order=None)[source]¶

Generate all independent variable combinations in a DataFrame.

Each row of the resulting DataFrame contains a unique combination of conditions. Use primarily for full counterbalancing of within-subject variables.

Parameters:
  • conditions – dict. Variable names and possible values.
  • order – list, optional. Order of columns in output.
Returns:

pandas.DataFrame. Each row is a unique combination of variables, assuming the possible values for each variable are unique.

 
my_variables = {'version':['a','b','c'], 'response':[1,2], 'session':2}
counterbalance(my_variables, order=['version','session','response'])
version session response

1 a 2 2
2 b 2 1
3 b 2 2
4 c 2 1

expand¶

labtools.trials_functions.expand(valid, name, values=[1.0, 0.0], ratio=0.5, sample=False, seed=None)[source]¶

Copy rows as necessary to satisfy the valid:invalid ratio.

Use when complete counterbalancing is not plausible. For example, when the ratio of trials requiring response A to those requiring response B is not 50:50.

Parameters:
  • valid – pandas.DataFrame. Trials to be expanded.
  • name – str. Name of new column containing valid and invalid values
  • values – list of length 2. Values for valid and invalid trials, respectively.
  • ratio – 0.0 < float < 1.0. Approximate percentage of valid trials in the resulting frame.
  • sample – bool, default False. Should the invalid trials be sampled from the valid trials? If True, len(returned) < 2*len(valid).
  • seed – float, optional. If sample is True, seed for random sampling of invalid trials.
Returns:

pandas.DataFrame. Valid and invalid trials are denoted in a new column.

 
posner = pd.DataFrame({'target_dir':['left','right']})
posner = expand(posner, 'valid_prime', ratio=0.75)

Shuffles trials a few times.

smart_shuffle¶

labtools.trials_functions.smart_shuffle(frame, col, block=None, seed=None, verbose=True, lim=10000)[source]¶

Shuffles trials such that equivalent trials never appear back to back.

Parameters:
  • frame – pandas.DataFrame of trials to be shuffled.
  • col – str column of values to minimize repetitions.
  • block – str column to groupby before shuffling.
  • seed – int, optional for repeatable randomization.
  • verbose – bool. Should the status of randomization be printed?
  • lim – int maximum number of shuffles before giving up.
Returns:

pandas.DataFrame with rows in randomized order.
